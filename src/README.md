# Intelligent Library Resource Manager 📚

A console-based Library Management System built using Java and Object-Oriented Programming principles.

## 🚀 Features

- Add new books
- View all available books
- Issue books
- Return books
- Real-time availability tracking
- Menu-driven console interface

## 🛠 Technologies Used

- Java
- OOP Concepts (Encapsulation, Abstraction)
- Java Collections (ArrayList)
- VS Code
- Git & GitHub

## ▶ How to Run the Project

1️⃣ Navigate to the src folder:

```
cd src
```

```
javac *.java
```

3️⃣ Run the project:

